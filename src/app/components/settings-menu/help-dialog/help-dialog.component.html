Info

ShowTunes is a web application that retrieves information from the Spotify API on the track currently playing on your Spotify account. It then updates and displays this information in a minimalistic and customizable web page with a fully controllable web player.

Feel free to take a look at the Github and open issues for any problems you find!

Features

  • Current track's album artwork
  • An optional Spotify code for the current track
  • Customization for the Spotify code's colors
  • Customization for the app's accent colors
  • A full music player to connect to your Spotify account
  • Light and dark themes

Help

  • dark_mode - Toggle between the light and dark themes
  • play_disabled - Turn the music player controls off
  • play_circle_outline - Turn the music player controls on, but fade after inactivity
  • play_circle_filled - Turn the music player controls on
  • queue_music - Toggle displaying the current playlist's name
  • qr_code_2 - Toggle the Spotify code on and off. A Spotify code is similar to a QR code and allows people to scan them to access the track on their Spotify app
  • lightbulb - Toggle display of settings for dynamic colors generated from the album's artwork
    • play_circle_outline - Toggle between using the dynamic color for the app's accent colors or a custom color/the default color
    • qr_code_2 - Toggle between using the dynamic color for the Spotify code or a fixed color
  • invert_colors - Toggle the bar code on the Spotify code as black or white. Only usable when dynamic color is set to off
  • Color picker - Set the background color of the Spotify code to a fixed color. The widget displays preset colors or allows a hex code to set to a specific color. This can be overridden by the dynamic code color
  • Accent color - Set the accent color of the app to a fixed color from the list rather than the default green color. This can be overridden by the dynamic accent color
