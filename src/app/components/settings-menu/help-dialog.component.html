Info

ShowTunes is a web application that retrieves information from the Spotify API on the track currently playing on your Spotify account. It then updates and displays this information in a minimalistic and customizable web page with a fully controllable web player.

Feel free to take a look at the Github and open issues for any problems you find!

Features

  • Current track's album artwork
  • An optional Spotify code for the current track
  • Customization for the Spotify code's colors
  • A full music player to connect to your Spotify account
  • Light and dark themes

Help

  • dark_mode - Toggle between the light and dark themes
  • play_circle - Toggle the music player controls on and off
  • queue_music - Toggle displaying the current playlist's name
  • qr_code_2 - Toggle the Spotify code on and off. A Spotify code is similar to a QR code and allows people to scan them to access the track on their Spotify app
  • lightbulb - Toggle between using a dynamic color generated from the current album's artwork for the Spotify code or a fixed color
  • invert_colors - Toggle the bar code on the Spotify code as black or white. Only usable when dynamic color is set to off
  • Color picker - Set the background color of the Spotify code to a fixed color. The widget displays preset colors or allows a hex code to set to a specific color. Only usable when dynamic color is set to off
